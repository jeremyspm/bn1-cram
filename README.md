# BN1 · 722.553 Foundations of Nursing — Active-Recall Cram

A single-file, offline study tool built around **one method only: active recall (retrieval practice)** — the highest-yield way to memorise for an exam. Every card makes you pull the answer from memory *before* it shows you anything, then schedules what you miss to come back fast (within-session Leitner spacing) and interleaves topics.

**▶ Live:** https://jeremyspm.github.io/bn1-cram/

## What's inside
- **669 cards across all 14 topic areas**, synthesized from the course lecture material and fact-checked, covering every subject on the exam blueprint.
- **180 "5-mark" structured cards** that mirror **Section B** exactly — recall the discrete marking points and tick off what you got. Section B is 70 of 112 marks.
- **🚨 Panic mode** — a 3-hour, **145 pass-critical card** blitz (the hand-triaged bare-minimum to scrape a pass, balanced so no topic is a blind spot).

## Modes
- **Smart mix** — adaptive, interleaved; let the engine drive.
- **Core blitz** — must-knows only.
- **5-mark drill** — Section B marking points.
- **Weak spots** — only what you're still failing.
- **🚨 I'm panicking** — the 3-hour pass blitz.

## How to use
Answer **out loud or on paper first — no peeking.** Then reveal and grade yourself honestly (Missed / Hard / Got it). Keyboard: **Space** reveal, **1 / 2 / 3** grade, **T** topics. Progress saves automatically on your device, so you can stop and resume. Tap **?** for the built-in 12-hour (and 3-hour panic) study plan.

## The exam it targets
722.553 Foundations of Nursing final: **42 MCQ** (1 mark each) + **14 short-answer** questions (5 marks each) = 112 marks.
